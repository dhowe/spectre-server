# SPECTRE
Worship at the Altar

## Installation

First make sure you have installed yarn (v1.16) for your OS, then:

```
git clone https://github.com/dhowe/spectre-server.git

cd spectre-server

yarn install
```

<br/>

### Test

```
yarn test
```

### Start (requires a local mongodb)

```
yarn start
```




<br/>

For more details on the client, see [these instructions](https://github.com/dhowe/spectre/)
